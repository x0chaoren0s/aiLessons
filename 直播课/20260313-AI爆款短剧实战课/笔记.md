
# 短剧优势
1. 碎片化：随时可看
2. 剧情爽：满足猎奇
3. 成本低、出片快，盈利空间大


# 专业短剧五步制作法
## 1. 得到想要的人物
    - 把想法告诉AI
    - AI生成图片
    - 选出最满意的图片
## 2. 让人物动起来
    - 形容要的动态
        - 简单动作：单图+提示词
        - 复杂动作：首尾帧+提示词
    - AI生成视频
    - 选出最满意的视频
## 3. 给人物添加剧情
    - 数字人
## 4. 完成原创剧情短剧
    - ai写大纲-分集-单集剧情-分镜
## 5. 连接所有分镜头
    - 剪映


# 如何收益
## 剧本
    - 按平台，一般千字15-1000元
    - 各大平台都在抢剧本，随便能搜到
    - 不会写？AI帮你写
    - 1.5w~5w / 部
    - 打款快
## 切片 
    - 账号收益+广告收益+佣金收益
    - 账号发布，账号带货
## 剧集 最大
    - 按集结算
    - 不同等级酬金不同+充值+分佣+广告+版权
