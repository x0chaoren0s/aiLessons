# 即梦AI实战：智能参考&风格转换综合应用


## 主题


### 即梦AI实战：智能参考&风格转换综合应用


## 本节课知识讲解


### 智能参考

#### 
```
图像编辑模型具备图片参考功能，
可以参考图片中的主体的特征、图片的色调、景深、风格、色彩、构图以及人物的姿势等等维度信息。
基于此功能，可以实现一句话编辑图像，如做产品设计、洗图等
```

#### 案例演示


##### 1.服装线稿上色设计


###### 提示词示例


   * 给图中的服装线稿上色，保持服装所占画面的比例和大小不变，优雅的红色连衣裙，写实，产品摄影


##### 2.室内设计


###### ai指令

```
你是室内设计专家，保持这个线稿不变，生成设计后的效果图。
现在需要你写简单的中文绘画提示词，描述这个房间各个家具的颜色和室内的风格。
中式风格和欧美风格各写一个提示词。
```

###### 提示词示例
```中式风格提示词
保持原图线稿布局，整体为新中式风格：
床头背景墙用深棕色木饰面配米白色硬包；
床品选浅灰色亚麻面料，搭配赭石色刺绣靠枕；
床头柜为黑胡桃木色，配黄铜拉手；
床尾凳是米色棉麻软包 + 深棕实木腿；
地面铺浅米色中式纹理地毯；
墙面刷暖白色乳胶漆，搭配水墨色调装饰画；
灯光用暖黄色，营造雅致温润的氛围。
```

##### 3.同款型不同配色的鞋子设计


###### 提示词示例
```
保持图中鞋子的款型和鞋子的轮廓不变，保持构图不变，生成四个新款的不同配色的鞋子。
```

##### 4.洗图


###### 什么是洗图？


   * 生成和参考图类似的一张图片，保留基本的构图、风格。生成的新图是自己的版权。


###### 示例提示词


   * 参考这个人物的姿势和造型以及视角，生成一个类似的模特


### 风格转换


#### 将参考图保留特征不变，转换一个新的风格


##### 支持的转换风格


###### 艺术与画作风格


   * 手绘彩色草稿、油画、厚涂、贴纸风格、黏土风格、Q版卡通插画、二次元、漫画风格


###### 摄影与写真风格


   * 复古港风、未来科技感/赛博朋克、极简风摄影、电影感、杂志大片、胶片质感


###### 文化特定风格


   * 古风、洛可可宫廷风、Y2K千禧辣妹风、清新风


###### 实体化风格


   * 商业化手办、拍立得照片


###### ......其他风格可以输入提示词尝试
