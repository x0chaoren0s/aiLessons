# 1. 节奏:内部节奏，外部节奏

## 1.1 什么是节奏?

- 节奏:元素呈现有规律的，可感知的变化形式。
- 节奏是电影的“心跳”和“呼吸”
- 电影节奏是导演通过综合艺术手段，引导和控制观众**心理时间**与**情感起伏**的艺术。
- 它不是物理节拍:不像音乐的节拍器那样有固定的“滴答"声，电影的节奏是可变的、流动的。
- 它是一种“感觉”:是你在观影时感到的“紧张得喘不过气”“悠闲自在”或“漫长煎熬”
- 它是时间的“雕刻师”:导演通过控制节奏，来“雕刻”观众的时间感。一部节奏出色的2小时电影，会让你觉得转瞬即逝;而一部节奏拖沓的90分钟电影，会让你如坐针毡.

## 1.2 内部节奏

- 这是由故事内容本身决定的。
- 情节节奏:故事的张弛变化。是危机四伏，还是闲庭信步?例如，侦探接近真相时的情节节奏必然加快。
    - 驱动源:矛盾冲突、悬念设置、信息揭示的速度。具体表现:
    - 快节奏情节:事件密集，冲突接踵而至，悬念迭起。例如《谍影重重》系列，主角始终处于被追杀的危机中，一个危机未平，另一个又起，几乎没有“喘息”的时间。这种节奏天然就带有紧张感和压迫感。
    - 慢节奏情节:事件稀疏，冲突缓和，悬念较少。例如《爱在黎明破晓前》三部曲，大部分时间是男女主角在城市中漫步、交谈，节奏舒缓，充满了生活气息和情感张力。
    - 变速节奏:大部分优秀作品都是变速的。遵循“紧张-松弛-再紧张”的规律，如同交响乐。例如《教父》，在血腥的暗杀之间，穿插着家族的婚礼、宁静的乡村生活，这种张弛有度使得高潮部分更具爆发力。

- 情绪/心理节奏:角色内心和观众感受的起伏。一段抒情戏可能节奏舒缓，一场生离死别则可能节奏沉重而强烈
    - 驱动源:角色的情感状态、心理活动和人物关系的演变。具体表现:
    - 角色内心平静时，情绪节奏是平缓的直线。当角色遭遇变故、陷人热恋、感到恐惧时，情绪节奏会产生剧烈的波动，
    - 关键在于，这种节奏是通过演员的表演和台词氛围传递给观众的，不依赖于剪辑。例如，在《小丑》的某些独角戏中，即使镜头一动不动，我们也能通过亚瑟·弗莱克从强颜欢笑到崩溃痛哭的表演，感受到他内心情绪节奏的崩塌。
    - 抒情节奏:情感舒缓，内心平静。例如《爱在黎明破晓前》中，男女主角在维也纳的漫步，充满了浪漫和诗意，节奏舒缓，让观众沉浸在美好的氛围中。
    - 紧张节奏:情感激动，内心冲突。例如《搏击俱乐部》中，主角的内心独白充满了焦虑和不安，节奏紧张，让观众感同身受。

- 对话节奏:台词的速度、密度和停顿。是艾伦·索金式的“子弹般”的快速对白，还是侯麦式的哲学性漫步对话?
    - 驱动源:对话内容、角色关系、戏剧张力。具体表现:
    - 快速对白:台词密集，语速快，停顿少。例如《社交网络》中，马克·扎克伯格与人对话时，语速极快，充满了思辨性和攻击性，营造出一种紧张的智力交锋氛围。
    - 舒缓对白:台词稀疏，语速慢，停顿多。例如《爱在黎明破晓前》中，男女主角的对话充满了哲学思考和情感交流，节奏舒缓，让观众沉浸在浪漫的氛围中。
    - 节奏的“呼吸感”：即使在对话场景中，节奏也需要呼吸。长时间的快速对白会让人疲惫，适时的停顿（如角色喝水、思考、眼神交流）能让观众喘口气，也能增强对话的真实感和张力。

- 内部节奏是指**单个镜头内部**的节奏感，主要通过以下元素控制：
- 景别：景别越小，节奏越快；景别越大，节奏越慢。
- 运镜：运镜越快，节奏越快；运镜越慢，节奏越慢。
- 画面构图：画面越复杂，节奏越快；画面越简单，节奏越慢。
- 画面内容：画面内容越丰富，节奏越快；画面内容越简单，节奏越慢。

## 1.2 外部节奏
核心定义:

外部节奏是通过一切影视技术手段，对外呈现出的、观众能直接感知到的视听律动。它是服务于内部节奏的“工具节奏”。

外部节奏是指**镜头与镜头之间**的节奏感，主要通过以下元素控制：

1. 剪辑节奏:最直接的操控手段.
镜头时长:
- 短镜头组合(高剪辑率):制造急促、混乱、兴奋或暴力感。如《疯狂的麦克斯4:狂暴之路》中的追车戏，大量一闪而过的镜头拼接，让观众眼花缭乱，血脉贲张。
- 长镜头(低剪辑率):营造沉浸、真实、压抑或诗意感。如《荒野猎人》中开场的长镜头战斗，一气呵成，让观众仿佛亲临其境，感受战斗的混乱与持
久
2. 摄影机运动节奏:镜头的“呼吸”
- 快速运动:急促的推、拉、摇、移，或手持跟拍，产生不稳定感和能量。如战争片中模拟士兵奔跑的主观镜头。
- 缓慢运动:平稳的轨道车移动、缓慢的横摇，营造宏大、庄严或凝重的氛围。如《银翼杀手2049》中许多缓慢移动的镜头，赋予了未来世界一种神性般的肃穆。
- 静止:固定机位本身也是一种节奏，它强调画面的构图和框内的信息，迫使观众静静观察。
3. 声音节奏:听觉的“脉搏”
- 音乐:节拍、旋律、配器直接影响情绪。激昂的管弦乐加快节奏，空灵的电子乐可能延缓节奏。
- 音效:心跳声、时钟滴答声、脚步声的密度和响度可以极大地强化紧张感。
- 静默:“此时无声胜有声”。突然的静默能制造巨大的悬念和压迫感，是节奏控制中的强力武器。

4. 表演与调度节奏:视觉的“韵律”
- 演员的肢体动作速度、走位速度也构成了外部节奏的一部分。一个角色缓慢地起身，与猛地跳起来，带来的节奏感是天差地别的。

- 镜头数量：镜头越多，节奏越快；镜头越少，节奏越慢。
- 镜头时长：镜头越短，节奏越快；镜头越长，节奏越慢。
- 镜头切换：镜头切换越频繁，节奏越快；镜头切换越不频繁，节奏越慢。
- 镜头运动：镜头运动越频繁，节奏越快；镜头运动越不频繁，节奏越慢。

# 2. 镜头切换的叙事功能
核心概念:每一次镜头切换，都不是随意的，而是一次有目的的叙事选择和对观众视线的引导。

1. 切
最常见的切换，瞬间从一个图像转到另一个。

叙事功能:
- 压缩时间/转换空间:最基础的功能。角色上车(切)角色下车，瞬间完成时空跳跃。
- 改变视点:从说话者(切)到倾听者的反应，这是对话场景的基石.建立关系:著名的“库里肖夫效应”:一个面无表情的脸(切)一碗汤(切)回脸，观众认为他饥饿;同一张脸(切)一个棺材(切)回脸，观众认为他悲伤。镜头A+镜头B，在观众心中产生了C(新的含义).这是蒙太奇理论的基石。
- 创造节奏:快速切换制造紧张，慢速切换营造宁静，

2. 淡入/淡出
- 画面逐渐出现或消失在黑场/白场。
- 叙事功能:表示一个时间段的开始或结束，带有强烈的段落感和抒情性。常用于影片开头、结尾或重大时间跳跃。

3. 溶
- 一个画面逐渐消失的同时，另一个画面逐渐显现，两者短暂重叠。
- 叙事功能:时间的柔和流逝:比“切"更诗意，表示一段时间过去了，关联与联想:将两个意象连接起来，产生比喻或象征意义。例如，溶从燃烧的火焰到飘扬的战旗，暗示革命的热情。

4. 划
- 一个新的画面从一侧“划"过，取代旧画面。
- 叙事功能:具有漫画感、活跃感和明确的转换提示，老电影中常用，现在多用于营造复古或诙谐的效果。

5. 匹配剪辑
- 一种高级的剪辑技巧，通过动作、图形或概念上的相似性进
- 叙事功能:
    - 动作匹配:上一个镜头人物抛出一个物体(切)下一个镜头另一个时空另一个人接住。流畅地连接不同时空，极具视觉趣味性(如《2001太空漫游》中骨头抛起变成太空船).
    - 图形匹配:利用形状的相似性切换(如圆形的方向盘切到圆形的蛋糕)
    - 概念匹配:通过意念连接(如一声尖叫切到鸣笛的火车)
